Для создания проекта Spring Boot с помощью Maven укажите в завимостях `ru.curs:spring-boot-starter-celesta`:

```xml
    <properties>
        <!-- dependencies -->
        <spring.boot.version>2.0.6.RELEASE</spring.boot.version>
        <spring.boot.starter.celesta.version>2.0.1-SNAPSHOT</spring.boot.starter.celesta.version>
    </properties>

. . .

    <dependencies>
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
            <version>${spring.boot.version}</version>
        </dependency>
        
        <dependency>
            <groupId>ru.curs</groupId>
            <artifactId>spring-boot-starter-celesta</artifactId>
            <version>${spring.boot.starter.celesta.version}</version>
        </dependency>
    ...
    </dependencies>

```

Также для кодогенерации <<data_accessors_section,классов доступа к данным>> понадобится `ru.curs:celesta-maven-plugin`. В его настройках необходимо указать путь к папке `score`:

```xml
    <build>
        <plugins>
            <plugin>
                <groupId>ru.curs</groupId>
                <artifactId>celesta-maven-plugin</artifactId>
                <version>7.0.4</version>
                <configuration>
                    <scores>
                        <score>
                            <path>${project.basedir}/src/main/resources/score</path>
                        </score>
                    </scores>
                </configuration>
                <executions>
                    <execution>
                        <goals>
                            <goal>gen-cursors</goal>
                        </goals>
                    </execution>
                </executions>
            </plugin>
        </plugins>
    </build>
```

В папке `src/main/resources/score` следует разместить скрипты определения базы данных на языке <<CelestaSQL,CelestaSQL>>. Например,

```sql
-- score/ru/curs/demo/demo.sql
CREATE SCHEMA demo VERSION '1.0';

CREATE TABLE OrderHeader(
  id VARCHAR(30) NOT NULL,
  date DATETIME,
  customer_id VARCHAR(30),
  customer_name VARCHAR(50),
  manager_id VARCHAR(30),
  CONSTRAINT Pk_OrderHeader PRIMARY KEY (id)
);
```
